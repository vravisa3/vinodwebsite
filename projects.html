Showcasing how data analytics and AI are transforming construction management.

ADOT Department Estimate Price Comparison Analysis

Research analyzes ADOT department estimate unit prices for seven common highway items across 23 projects, revealing extreme cost variability driven by project-specific conditions and recommending risk-band based, data-driven estimating practices to improve consistency and budgeting reliability.

Tools: AI, Excel

Type: Academic Project

IoT‑Based Smart Slope Stability Monitoring and Landslide Early Warning System

ARAM (Always Ready Avoiding Mishaps) is a smart landslide early‑warning system using IoT soil, tilt, vibration, and weather sensors to monitor slope stability in real time. It was prototyped with Arduino testing and expert input, showing fast, accurate alerts that help construction teams evacuate early, reduce inspections, and prevent costly slope failures.

Type: Academic Project

Pavement Performance Prediction Modelling for Urban Roads in Coimbatore City

The project develops a statistical model to predict flexible pavement condition on Coimbatore urban roads using traffic, structural, and distress data. It evaluates selected road stretches, rates performance via condition indices, and identifies key deterioration factors, enabling targeted maintenance planning, optimized budgets, and improved service life for urban pavements.

Tools: Python, Machine Learning, Data Analytics

Type: Academic Project

IoT Based Car Parking System using LoRaWAN Technology for CIT Campus

IoT-based smart parking system for CIT campus using LoRaWAN to monitor real-time slot availability and guide drivers, reducing congestion and search time. Scope covers sensor deployment, network design, and campus integration. Built with IoT nodes and gateway communicating via LoRaWAN; results show improved utilization and efficient space management.

Type: Academic Project
